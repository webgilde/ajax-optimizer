Allows to disable selected plugins in AJAX requests initiated by the frontend

== Description ==

By default, WordPress loads all plugins during AJAX requests initiated by the frontend. This plugin allows to disable some of them.

== Installation ==

1. Upload `ajax-optimizer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Plugins > AJAX Optimizer` to set up settungs
